# Hurtz Publicações

Páginas públicas da Hurtz Company — propostas, landings, materiais de marketing.

**URL:** https://pages.hurtzcompany.com.br

## Estrutura

Cada pasta vira um slug na URL:

```
teste/          → /teste/
proposta-x/     → /proposta-x/
landing-y/      → /landing-y/
```

## Como adicionar página

1. Criar pasta: `nome-da-pagina/`
2. Criar `index.html` dentro
3. Git commit + push
4. Deploy automático via Cloudflare Pages

## Deploy

Conectado ao Cloudflare Pages via GitHub. Deploy automático a cada push.
