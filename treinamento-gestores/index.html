HURTZ CÉREBRO 05 · SYNC GIT
05 · DEPLOY / SYNC

Git acontece sozinho.

Você nunca digita git commit ou git push. Hooks rodam automaticamente em cada sessão. O que você escreve hoje, os outros gestores veem amanhã.

QUANDO ABRE

SessionStart hook

git pull --rebase --autostash

Puxa tudo que outros gestores gravaram desde a última vez. Silencioso, alerta só em conflito.

DURANTE

Você fala. Claude grava.

Cada vez que você pede algo que altera arquivo, Claude edita. As mudanças acumulam localmente.

Tu vê na aba Source Control (Ctrl+Shift+G) o que mudou.

QUANDO FECHA

Stop hook

git add → commit → pull → push

Tudo é versionado e enviado pro GitHub. Sai e fecha o computador. No segundo seguinte, os outros gestores recebem.

Quer forçar o sync agora? Pede /syncar no Claude. Ele faz add + commit + push imediato. Útil antes de uma reunião quando você acabou de gravar algo importante.
Conflito? Quando dois gestores editam a mesma linha do mesmo arquivo. Raríssimo (a gente configurou áreas append-only pra evitar). Quando rolar, Claude avisa e ajuda a resolver.
