HURTZ · 04 / 15
HURTZ CÉREBRO 04 · SETUP INICIAL
04 · SETUP

30 minutos. Uma vez só.

01.

Instalar Claude Code

Terminal aberto, roda o curl. Login com a conta da Hurtz.

$ curl -fsSL claude.ai/install.sh | bash
02.

Clonar o repo

Acesso ao repo o Klebson libera. Clone numa pasta local.

$ git clone hurtzgestao-lang/
hurtz-consorcio.git
03.

Configurar .env

Copia o template, ajusta teu slug e segredos do Klebson.

$ cp .env.example .env
# HURTZ_GESTOR=seu-slug
04.

Rodar /iniciar

Claude lê o contexto da empresa + teu agente pessoal.

$ claude
› /iniciar

HURTZ · 12 / 15
HURTZ CÉREBRO 12 · GUARDRAILS
12 · O QUE NUNCA FAZER

5 coisas pra nunca fazer.

Editar pasta de outro gestor

colaboradores/{outro}/ é privado dele. Use /criar-tarefa pra adicionar coisas no inbox.

Apagar arquivo de cliente

Pede pro Claude mover, nunca deletar. Histórico do cliente é sagrado.

Commitar .env ou credenciais

Já tá no .gitignore — mas confere antes de qualquer push manual.

git push --force

Reescreve histórico da equipe. Se precisar, chama o Klebson.

Editar metodologias / estratégia direto

base-conhecimento/metodologias/ e empresa/estrategia.md só entram com aprovação do Klebson e Marcos.
